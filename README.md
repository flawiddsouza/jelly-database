## Setup

```
cd ui
npm i
npm run build
cd ..
npm install
```

## To Run

```
npm start
```

## Quick install on distributions with systemd:

```
curl -s https://raw.githubusercontent.com/flawiddsouza/jellyadmin/main/install.sh | bash
```

## Common issues

### MySQL import doesn't work

```
sudo apt-get install mysql-client
```
